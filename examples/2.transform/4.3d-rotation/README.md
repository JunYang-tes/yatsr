## 三维空间的基本旋转

![rotation](./rotation.svg)

三维空间有三种基本的旋转，分别为绕三个轴的旋转。上图展示了绕三个轴旋转的正方向。z轴垂直于屏幕（或渲染图片）向外。

### 绕z轴旋转

绕z轴旋转时，z坐标保持不变。x,y坐标的旋转方式和2D空间相同。
